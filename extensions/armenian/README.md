# Armenian - Code Spell Checker

Armenian dictionary extension for VS Code.

This is an add-on for the [Code Spell Checker](https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker).

## Requirements

Requires [Code Spell Checker](https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker) extension.

## Usage

After installing, the Armenian dictionary will be automatically enabled for files with Armenian text.

You can also manually enable or disable it using the following commands:

- `Enable Armenian Spell Checker Dictionary`
- `Disable Armenian Spell Checker Dictionary`

## Support

- Armenian (`hy`)
- Armenian (Armenia) (`hy-AM`)
